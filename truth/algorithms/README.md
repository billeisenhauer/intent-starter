# Algorithms

Computation rules that define system behavior.

## Required Content

- Formulas
- Weights
- Thresholds
- Decay functions
- Scoring parameters

Each algorithm must include:
- Name
- Formula or rule
- Parameters with values

## Forbidden Content

- Implementation code
- Performance optimizations
- Language-specific syntax
- Rationale prose (use parameter tables)

If a number affects system behavior,
it belongs here.

If a number affects only performance,
it does not.
