# Skills

Skills are reusable prompts that guide AI behavior for specific tasks.

## Available Skills

| Skill | Command | Purpose |
|-------|---------|---------|
| Intent Distillation | `/intent-distill` | Extract truth artifacts from PRDs and stories |
| Boundary Classification | `/boundary-classify` | Classify changes by pace layer |
| Truth Verification | `/truth-verify` | Run verification with formatted summary |

## How Skills Work

Skills provide structured guidance for common operations.
They ensure consistent behavior across sessions.

## Skill Categories

### Before Coding
- `/boundary-classify` — Know what layer you're touching

### During Development
- `/intent-distill` — Convert requirements into truth artifacts

### Before Committing
- `/truth-verify` — Ensure system identity is preserved
